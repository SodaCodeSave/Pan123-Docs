## 版本号说明

Pan123 遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- `MAJOR.MINOR.PATCH`
  - `MAJOR` 版本号：当进行不兼容的API修改时
  - `MINOR` 版本号：当以向后兼容的方式添加功能时
  - `PATCH` 版本号：当进行向后兼容的问题修复时

## 预发布版本

- `alpha` - 内部测试版本，功能不完整
- `beta` - 公开测试版本，功能完整但可能有bug
- `rc` - 发布候选版本，准备正式发布

## 更新方法

### 升级到最新版本

```bash
pip install --upgrade pan123
```

### 安装特定版本

```bash
pip install pan123==0.1.9
```

### 查看当前版本

```bash
pip show pan123
```

## 升级注意事项

### 从 v0.1.x 升级到 v0.2.x

> ⚠️ **重要**：v0.2.x 版本包含重大变化，请在升级前阅读升级指南。

- 异步API变更：某些同步方法可能变为异步
- 参数变更：部分方法的参数可能有所调整
- 返回值变更：API返回值格式可能有所变化

### 从 v0.1.7 升级到 v0.1.8

- OSS模块：OSS模块中的离线下载功能已弃用，请使用主模块的离线下载功能
- 认证修复：修复了get_access_token函数的问题，现在应该正常工作

## 贡献版本历史

如果要为 Pan123 贡献代码，请遵循以下提交信息格式：

